# learning-AI/ML-Project-Image Classification
## Overview
This project implements an image classification pipeline using Python and Jupyter Notebook. It was developed and tested as a part of my learning in machine learning and computer vision.
## DataSet
Name: Breast Cancer Histopathological Images(BreakHis)
Source: Kaggle
DataSet link: https://www.kaggle.com/datasets/kritika397/breast-cancer-dataset-from-breakhis
This dataset is not included in this repository due to size and licensing constraints.
Users are expected to download the dataset separately from Kaggle.
## Tools and Technologies
Python
Jupyter Notebook
PyTorch
EfficientVit
## How to Run
Download the dataset from Kaggle(link below).
Place the dataset in the directory structure mentioned in the notebook.
Run the notebook cells sequentially.
